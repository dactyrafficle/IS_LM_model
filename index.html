    What is epsilon? And what is theta?

    If L = A Y^{\epsilon}, then \frac{dL}{dY} = \epsilon A Y^{\epsilon-1} = \epsilon A Y^{\epsilon} Y^{-1} = \epsilon L Y^{-1}

    So that means \dfrac{dL/L}{dY/Y} = \frac{dL}{dY} \frac{Y}{L} = \epsilon L Y^{-1} \cdot Y L^{-1} = \epsilon

    If Y_2=1.01 \cdot Y_1, then \Delta Y = Y_2-Y_1=0.01Y_1

    So \dfrac{\Delta L/L}{\Delta Y/Y} = \dfrac{\Delta L/L}{\Delta 0.01 \cdot Y_1/Y_1} = \dfrac{\Delta L/L}{0.01} = \epsilon

    Meaning \Delta L/L = 0.01 \cdot \epsilon

    So a 1% change in Y means an ε % change in L

    Conversely, a 1% change in r means a -θ % change in L
